# 🌙 محفل - راهنمای نصب و راه‌اندازی

## پیش‌نیازها
- **Node.js** نسخه 18 یا بالاتر → https://nodejs.org

---

## مراحل راه‌اندازی

### ۱. باز کردن ترمینال
روی ویندوز: `Win + R` → `cmd` یا PowerShell

### ۲. رفتن به پوشه پروژه
```bash
cd mahfel
```

### ۳. نصب وابستگی‌ها (فقط دفعه اول)
```bash
npm install
```

### ۴. اجرای سرور
```bash
node server.js
```

خروجی باید اینو نشون بده:
```
🚀 محفل روی http://localhost:3000
```

### ۵. باز کردن سایت
مرورگر رو باز کن و برو به:
```
http://localhost:3000
```

---

## دوستات چطور جوین بشن؟

### روی همون شبکه (WiFi مشترک):
1. آدرس IP خودت رو پیدا کن:
   - ویندوز: `ipconfig` توی cmd
   - مک/لینوکس: `ifconfig` یا `ip addr`
2. آدرس رو بده به دوستت: `http://192.168.x.x:3000`

### از اینترنت (راه سریع):
1. نصب ngrok: https://ngrok.com
2. بعد از اجرای سرور، بزن:
   ```
   ngrok http 3000
   ```
3. لینک عمومی که ngrok میده رو بده به دوستات
